HTML < section > Element


A section in HTML is a semantic element that is used to group together related content on a web page. It is typically used to divide the page into sections, with each section having its own heading. The < section > tag is often used when creating a landing page to divide the page into separate logical blocks.


According to the W3C specification, a section is a thematic grouping of content, typically with a heading 1. The < section > tag is used to define a section in an HTML document 1. It is a block-level element that can be used to group together related content 1. Sections should always have a heading, with very few exceptions
